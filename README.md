# EyeControlMac

## Preview Testing

For Preview testing, use View > Single Page or View > Two Pages if Next/Previous Page does not visibly move pages.

## Manual Scroll Testing

1. Run EyeControlMac from Xcode.
2. Grant Accessibility permission to EyeControlMac and Xcode.
3. Open a PDF in Preview.
4. Hover the mouse over the PDF page body.
5. Set EyeControlMac to Active.
6. Select Scroll Down or Scroll Up.
7. Press Space.
